# Customization

## Content
- **Eleventy**: Update text and data in `frontend-11ty/src/data/` and templates in `frontend-11ty/src/` (e.g., `index.md`, components, and includes).
- **React**: Edit content and section data in `frontend-react/src/data/` and update components in `frontend-react/src/components/`.

## Styling
- **Tailwind**: Adjust Tailwind utilities or add new styles in `frontend-11ty/src/input.css` or `frontend-react/src/index.css`.
- **SCSS (Eleventy only)**: If you use the optional SCSS pipeline, edit files under `frontend-11ty/src/assets/scss/` and run `npm run sass` to compile.

## Assets
Replace images or other static assets in the corresponding `frontend-11ty/src/assets/` or `frontend-react/src/assets/` directories.
